# 学生管理系统（Student Management System）

## 📌 项目简介

学生管理系统是一个基于**前后端分离架构**的教学管理类 Web 应用，模拟高校场景下学生、教师、辅导员与超级管理员之间的核心业务流程，重点体现**角色权限控制（RBAC）**与清晰的业务边界设计。

本项目适用于课程设计、综合实践及个人全栈项目展示。

---

## 🎯 项目目标

- 构建一个具备真实业务逻辑的学生管理系统
- 明确多角色权限边界与职责划分
- 实现学生与专业班级、课程班级之间的管理关系
- 提供结构清晰、可扩展的系统设计方案

---

## 👥 系统角色

- **学生**：维护个人信息，加入或退出班级  
- **教师**：管理所负责课程班级的学生  
- **辅导员**：管理所负责专业班级的学生  
- **超级管理员**：系统级管理者，拥有全部权限  

---

## ✨ 核心功能

- 多角色登录与权限控制  
- 学生账号注册 / 创建 / 登录 / 密码管理  
- 专业班级与课程班级管理  
- 学生与班级关系管理  
- 按角色的一键查询功能
